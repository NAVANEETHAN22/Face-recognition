# 🎓 Smart Attendance System Using Face Recognition

A web-based smart attendance management system that automates attendance tracking using facial recognition technology. The system captures faces in real time, compares them with a predefined dataset, and records attendance automatically in an Excel sheet.

This project was developed as a team effort involving three members and was later published as a research paper in the EAI Endorsed Transactions on Scalable Information Systems.

## 🎯 Project Overview

Traditional attendance systems are time-consuming and prone to errors. This project automates the attendance process using facial recognition technology.

The system:

- Detects faces from a live camera feed.
- Extracts facial features.
- Compares detected faces with a predefined dataset.
- Recognizes individuals in real time.
- Automatically marks attendance.
- Stores attendance records in an Excel file.

---

## ✨ Features

### 👤 Face Detection
- Detects human faces in real-time video streams.
- Supports multiple face recognition operations.

### 🧠 Facial Feature Extraction
- Extracts facial embeddings using Face Recognition Library.
- Stores facial encodings for future comparisons.

### 🎥 Real-Time Recognition
- Live webcam-based face recognition.
- Fast and efficient face matching.

### 📊 Attendance Management
- Automatically records attendance.
- Saves attendance date and timestamp.
- Exports attendance records to Excel.

### 🌐 Web Application
- Built using Flask.
- Easy-to-use browser interface.

## ⚙️ System Workflow

1. Create a predefined dataset of student images.
2. Extract facial features from each image.
3. Store facial encodings in memory.
4. Capture real-time video using webcam.
5. Detect face locations.
6. Convert image from BGR to RGB.
7. Generate facial encodings.
8. Compare detected faces with stored encodings.
9. Mark attendance automatically.
10. Save attendance details into Excel.
